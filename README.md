#hitlib2

##这是一个现代版的哈工大图书馆网站

我想让学校图书馆的领导知道，网站这种东西想做的好看一点，其实也没那么难。哈工大是一流的学校，工大图书馆也是一流的图书馆，请不要再让我们忍受一个不入流的网站。
